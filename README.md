# customer_behaviour-
This project was completed as an independent academic and practical learning initiative. Throughout the development process, I leveraged technical documentation and structured online learning resources to enhance my understanding of analytical techniques and visualization standards.

Objective:
1. Analyze customer subscription distribution
2. Identify revenue trends across product categories
3. Evaluate sales performance across age groups
4. Examine average purchase amount and customer ratings
5. Develop an interactive dashboard for business insights

tools i used:
jupyter notebook- numpy, pandas
postgresql (pgAdmin4) for sql queries
power BI

Business Impact
This analysis helps businesses:
Improve customer segmentation strategies
Optimize high-performing product categories
Identify revenue-driving customer groups
Support strategic and data-informed decisions

This project was developed as part of my hands-on learning in data analytics and business intelligence. During development, I referred to official documentation and structured educational resources to strengthen conceptual understanding and ensure adherence to best practices.
